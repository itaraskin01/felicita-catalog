## Что сделать тебе (3 шага)

1. **Захостить папку `catalog_2026/`** в интернете (см. подробности в `../catalog/README.md`). Самый быстрый путь — публичный репозиторий GitHub.

2. **Заменить плейсхолдер** `https://YOUR-HOST.example.com/felicita/catalog_2026` на реальный URL в `catalog.yml`:
   ```bash
   cd "/Users/itaraskin/Desktop/Felicita/vkontakte upload/catalog_2026"
   sed -i '' 's|https://YOUR-HOST.example.com/felicita/catalog_2026|https://raw.githubusercontent.com/<твой-логин>/felicita-catalog/main/catalog_2026|g' catalog.yml
   ```

3. **В VK** «Импорт товаров из файла» → вставить публичный URL `catalog.yml` → «Добавить товары».
   **Перед загрузкой удалите старые товары** из VK Market, иначе появятся дубли (или назначайте offer id с префиксом, чтобы перетереть).

## Замечания

- Пять-семь товаров (FEL-0001..0004 и пара других) — это анонсы коллекций без явного «арт XXX — Y руб». Парсер их зацепил, но название корявое. При желании удалю их вручную, либо удали сам прямо в VK после импорта.
- В постах со «скидка 50% — было 18 900 / стало 9 450» проставлены `<oldprice>` и `<price>` корректно.
- Категория `13 Прочее` нигде не использована — все товары попали в свои разделы (Туфли/Ботинки/Босоножки/Лоферы/Кроссовки/Сумки/Рюкзаки).
